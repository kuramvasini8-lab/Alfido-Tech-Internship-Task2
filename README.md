# 🧠 Web Development Quiz App

An interactive and responsive **Quiz Application** developed using **HTML, CSS, and JavaScript DOM Manipulation**.
The application tests users on Web Development concepts through multiple-choice questions with a timer, instant feedback, score tracking, and animated UI interactions.

---

# 🎯 Objective

The main objective of this project is to build a **mini interactive JavaScript application using DOM manipulation**.
This project was created to improve practical knowledge of:

* JavaScript DOM Manipulation
* Dynamic UI Rendering
* Event Handling
* Responsive Web Design
* Interactive User Experience

# 📌 JavaScript DOM Features Used

This project heavily focuses on **DOM Manipulation** using JavaScript.

### DOM Concepts Implemented:

* `document.getElementById()`
* `document.querySelector()`
* `document.createElement()`
* `appendChild()`
* `innerHTML`
* `textContent`
* `classList.add()`
* `classList.remove()`
* Event Handling using `onclick`
* Dynamic Question Rendering
* Dynamic Option Generation
* Timer Updates using `setInterval()`
* Screen Navigation using DOM
* Real-time Score Updates
